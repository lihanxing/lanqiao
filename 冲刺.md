## 成绩分析
![](https://github.com/lihanxing/lanqiao/blob/%E5%9B%BE%E7%89%87/QQ%E5%9B%BE%E7%89%8720210412102606.png)
* 四射五入按一定位数输出
```
n = int(input())

ans = []
for i in range(n):
    temp = int(input())
    ans.append(temp)

print(max(ans))
print(min(ans))
print(round((sum(ans))/n,2))

```

## 小数第n位
* http://lx.lanqiao.cn/problem.page?gpid=T456
```
a,b,n = map(int,input().split())
for i in range(n+2):
    a=a*10
    if i>=n-1:
        print(a//b,end = '')
    a = a%b

```
