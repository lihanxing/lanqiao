## 合数个数
* 定义判断素数的函数
```
import math
def is_prime(n):
    sqrt = int(math.sqrt(n))

    for i in range(2,sqrt+1):
        if n%i==0:
            return False
    
    return True

count = 0

for i in range(2,2021):
    if is_prime(i):
        count+=1
    

print(2019-count)
```
