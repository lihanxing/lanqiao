## 核桃的数量
* map函数的用法
* 定义函数去解决问题
* 求最小公倍数
```python
#输入一行数字，每个数字间带空格
a,b,c = map(int,input().split())

def zuixiao(a,b):
    n = min(a,b)
    while True:
        if n%a == 0 and n%b == 0:
            return n
            break
        n += 1
    

ans = zuixiao(c,zuixiao(a,b))
print(ans)


```


## 不同子串
* 没啥难点
```python
result = []
strs = '0100110001010001'
n = len(strs)
for i in range(n):
    temp = strs[i]
    if temp not in result:
        result.append(temp)
for i in range(1,n+1):
    j = 0
    while (j+i)<n:
        temp = strs[j:j+i+1]        
        if temp not in result:
            result.append(temp)           
        j+=1

print(len(result))
```

## 数的分解
* 认为i,j,k为上升序列
```
def test(n):
    n = str(n)
    if '2' in n:
        return False
    elif '4' in n:
        return False
    
    return True


count = 0
for i in range(1,2019):
    if test(i):
        for j in range(i+1,2019):
            if test(j):
                k = 2019-i-j
                if k>j and test(k):
                    print(i,j,k)
                    count+=1

print(count)
```
## 特别数的和

        return True
