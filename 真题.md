## 核桃的数量
* map函数的用法
* 定义函数去解决问题
* 求最小公倍数
```python
#输入一行数字，每个数字间带空格
a,b,c = map(int,input().split())

def zuixiao(a,b):
    n = min(a,b)
    while True:
        if n%a == 0 and n%b == 0:
            return n
            break
        n += 1
    

ans = zuixiao(c,zuixiao(a,b))
print(ans)


```


## 不同子串
* 没啥难点
```python
result = []
strs = '0100110001010001'
n = len(strs)
for i in range(n):
    temp = strs[i]
    if temp not in result:
        result.append(temp)
for i in range(1,n+1):
    j = 0
    while (j+i)<n:
        temp = strs[j:j+i+1]        
        if temp not in result:
            result.append(temp)           
        j+=1

print(len(result))
```

## 年号子串
* 没啥难度
```python

strs = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
result = []
n = int(input())

a = int(n//26)
n = n%26

if a>0:
    result.append(strs[a-1])

result.append(strs[n-1])

for i in result:
    print(i,end='')
```
